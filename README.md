## 💡 About
FluexGL DSP is fully channel-based, and this application provides a clear, hands-on look at how it works in practice.  
The interface is inspired by FL Studio, with some of its recognizable color choices reflected in the design.

## 💻 How to use
Clone the project and install the dependencies using `npm install`.  
Once the installation is complete, run `npm run dev` to launch the Vite development server.

The information header inside the application will guide you through the basic workflow.

## 🎚️ Technologies used
This mixer is built using the following technologies:
- FluexGL DSP
- Vite
- Rust bindgen
- React
- TypeScript

## ⚠️ Disclaimer
No UI library was used in this application. Everything was designed by hand, and no AI was involved in creating the core of this project.*

This application is no longer actively maintained.  
For the latest and fully functional version, check out the live demo here:  
👉 https://www.fluexgl.dev/dsp/mixer
